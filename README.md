# vaadin-date-range-picker
A Vaadin Flow DateRangePicker implementation

![demo](demo.png)

## Run the Demo
* Checkout the repo
* Run ``mvn clean package``
* Navigate into ``vaadin-date-range-picker-demo`` 
* Run ``mvn jetty:run``
* Open http://localhost:8080
